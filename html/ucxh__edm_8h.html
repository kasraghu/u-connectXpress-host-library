◆ edmParse()

int edmParse ( char *  pInData,
unsigned int  size,
ucxhEDM_event_t *  pEventData,
unsigned int *  pConsumed 
)

Function for parsing binary EDM data into the event structure.

A complete EDM binary packet must be present in the buffer. To guarantee that a packet can be fit in the buffer, the buffer must be ucxhEDM_MAX_OVERHEAD + ucxhEDM_MAX_SIZE bytes. The first byte must contain the ucxhEDM_SOH, and the last byte of the packet must be ucxhEDM_EOT

Note
Only the first found EDM packet is parsed. So (after handling the event data output) rerun the function until all data is consumed.
Special case ucxhEDM_ERROR_INCOMPLETE in which case all data has probably not been received yet.
Parameters
[in]pInDataPointer to the binary data to parse
[in]sizeSize of the binary data. Must be ucxhEDM_MIN_SIZE bytes or larger.
[out]pEventDataOutput structure, needs to be allocated by caller before call. Only valid if result code is ucxhEDM_OK.
[out]pConsumedThe number of bytes of the input that has been parsed. See function description on how to handle.
Postcondition
If output type is ucxhEDM_EVENT_DATA or ucxhEDM_EVENT_AT the data/command pointer in pEventData points to memory in the original inData. So inData most still be in scoop when pEventData is used.
Return values
ucxhEDM_OKSuccessful parsing
ucxhEDM_ERROR_PARAMinData is NULL or size in invalid
ucxhEDM_ERROR_INVALIDNot a valid EDM packet
ucxhEDM_ERROR_INCOMPLETEStart of a valid EDM packet, but does not contain a full packet
ucxhEDM_ERROR_SIZEEDM size header does not match actual data, should never happen
ucxhEDM_ERROR_CORRUPTEDInvalid value of data inside the EDM packet, should never happen

Function packing an AT command request into an EDM packet.

Note
This function will include a memcpy of size bytes, if there is desire to avoid this for performance and/or memory reasons use the edmZeroCopy functions instead.
Parameters
[in]pAtStringPointer to the AT Command, must end with CR (0x0D).
[in]atStringSizeSize of the AT Command not including terminating character. Must be less than ucxhEDM_MAX_SIZE.
[out]pEdmPacketPointer to a memory where the EDM packet is created. This need to be an allocated memory area of AT request size + ucxhEDM_REQUEST_OVERHEAD.
Return values
positiveNumber of bytes used in the packet memory. Typically size + ucxhEDM_REQUEST_OVERHEAD.
ucxhEDM_ERROR_PARAMInput pointer and null or size is to large.

int edmZeroCopyHeadRequest ( int  atStringSize,
char *  pHeadData 
)

Creates an EDM AT request packet header.

This is a way to avoid a memcpy and can e.g. be useful if the packet is sent over a stream based channel the accepts bytes/chunks. The module will assemble the input and execute when a full EDM packet is received.
Valid EDM packet: head + AT request + tail.

Parameters
[in]atStringSizeSize of the AT request.
[out]pHeadDataPointer to a memory where the EDM packet is created. This need to be an allocated memory area of ucxhEDM_REQUEST_OVERHEAD.
Return values
positiveNumber of bytes used in the head memory.
ucxhEDM_ERROR_PARAMInput pointer and null or size is to large.
