◆ ucxhCOMMAND_writeWifiStationConfiguration()

bool ucxhCOMMAND_writeWifiStationConfiguration ( const ucxhPARSER_wifiStationConfigurationParam_t *  param)

Define and write a configuration parameter to RAM for the WiFi Station.

Note
Does not store the configuration to NVRAM or activate the WiFi Station.
To write several parameters at once (recommended), use ucxhMODULE_writeWifiStationConfiguration() instad. To optionally store the configuration to NVRAM, use ucxhMODULE_storeWifiStationConfiguration(). To activate the WiFi Station, use ucxhMODULE_activateWifiStation().
Related AT-command:
  • AT+UWSC
See also
ucxhMODULE_storeWifiStationConfiguration()
ucxhMODULE_resetWifiStationConfiguration()
ucxhMODULE_activateWifiStation()
ucxhMODULE_deactivateWifiStation()
Warning
The Wifi Station must not be activated when executing this command.
