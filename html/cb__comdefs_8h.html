◆ cb_BIT_0

#define cb_BIT_0   (1ul)

Used in function definitions to declare an input parameter unused to avoid warnings. Used when declaring an empty array that does not take up space in a struct. Example: struct { cb_uint8 payload[cb_EMPTY_ARRAY]; } In some compilers this is empty i.e. payload[]. While in some it requires a zero. I.e. payload[0]; Use this define to get it working for your system.
